Два моста

Arbitrum Bridge

Актив: USDC
Депозит: ~10-15 мин (retryable ticket)
Вывод: 7 дней (или fast 0.3% fee → моментально)
Challenge window: 7 дней
Использование: Основной мост для крупных сумм

TON Bridge

Актив: USDT
Депозит: ~3 сек
Вывод: ~1 сек (мгновенно)
Challenge window: Нет
Использование: Telegram Mini App, retail

⚠ Оба моста — in-memory mock. Реальная интеграция с Arbitrum Nitro и TON RPC — следующий шаг.

Telegram Mini App

  • Компактный UI внутри Telegram
  • Авторизация через WebApp SDK (HMAC initData)
  • Депозит USDT (TON) → торговля → вывод USDT (TON)
  • Пользователи tg-* пропускают RSA-верификацию
  • Падение если не в Telegram — грациозное

Файлы: ui/telegram.html · ui/telegram-app.js · ui/style.css

Экономика

Доход

Protocol fee: 0.02-0.05%

С каждой сделки. Без нативного токена.

  • Fast withdrawal fee: 0.3% → LP
  • Funding rate: переменная (premium index)
  • Возможен native token: стейкинг, DAO, fee discount

Расходы

~$600/мес

  • 3 × bare metal VPS ($30-50/мес each)
  • Arbitrum batch: ~$17-50/мес (@ $0.001-0.003)
  • TON RPC: ~$0 (public endpoints)
  • Домены, SSL: Free (Let's Encrypt)

PnL положительный при ежедневном объёме > $100k (при fee 0.03%)

Регуляторная стратегия

  1. Incorporation: Cayman Islands / BVI
  2. Компания владеет IP, но не управляет биржей
  3. Сеть валидаторов — открытая, код open-source
  4. Геоблокировка US + sanctioned countries (Cloudflare IP)
  5. Без KYC — кошелёк = адрес, торговля без идентификации
  6. Аргумент регулятору: «мы — протокол (L2 settlement layer), не биржа. Код открыт, валидаторы независимы»

⚠ Это не юридическая консультация. Требуется advice от юриста по crypto regulation.
